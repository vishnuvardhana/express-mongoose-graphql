# graphql-nodejs-mongodb
Simple starter kit for grapql, mongoDB with mongoose with express js.


#Installation and usage

1. Clone the repo
2. cd graphql-nodejs-mongodb
3. yarn install
4. node server.js 
5. go to http://localhost:4000/graphql
